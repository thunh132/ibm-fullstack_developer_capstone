# IBM Full Stack Developer Capstone

## Repository Name
ibm-fullstack_developer_capstone

## Project Name
Car Dealership Review Application

## Description
A full-stack web application built with Django and React that allows users to view car dealerships, read reviews, and submit their own reviews with sentiment analysis.

## Technologies Used
- Backend: Django 5.0.3
- Frontend: React.js
- Database: SQLite
- Sentiment Analysis: Python NLP

## Features
- View all car dealerships
- Filter dealers by state
- View dealer reviews
- Submit reviews with sentiment analysis
- User authentication (Register, Login, Logout)
